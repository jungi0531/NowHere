# PRD

## Product

NowHere is a mobile product that generates a personalized meditation session from a user's daily check-in and delivers it as a TTS audio file.

## Core Thesis

Users are more likely to return to a meditation app when the content reflects their current emotional context instead of replaying generic static content.

## Primary Flow

1. User completes a short daily check-in.
2. The system generates a meditation script.
3. The system generates an audio file from that script.
4. The user listens to the session.
5. The app stores the session result and lightweight reflection.

## MVP Scope

- daily check-in
- meditation script generation
- TTS audio file generation
- audio playback
- session log
- lightweight retention signals

## Out of Scope

- real-time conversational voice coach
- heavy gamification
- social features
- long-form journaling analysis
- character-centric interaction systems

## MVP Success Signals

- check-in completion rate
- session generation success rate
- playback start rate
- playback completion rate
- weekly repeat usage

## Product Constraints

- audio is file-based, not live-stream conversational audio
- latency is acceptable if the result feels complete and predictable
- the system should remain simple enough for a solo or small-team workflow
